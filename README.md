# SizeKit

一个静态的设计尺寸画板工具。

## 运行

### 单文件版
直接双击 `sizekit-single.html`。

### 多文件版
保持以下文件在同一个目录，然后双击 `index.html`：

```text
index.html
styles.css
app.js
```

也可以用 VS Code 的 Live Server 打开 `index.html`。

## 当前交互

1. 左侧窄工具栏只显示平台/物料图标。
2. 点击图标后，右侧画布显示该平台/物料的全部尺寸。
3. 每个尺寸以对应比例的画板矩形展示。
4. 点击画板查看详情，双击画板复制尺寸。
5. 平面印刷尺寸支持 72 / 150 / 300 dpi 像素换算。
